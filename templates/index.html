ShadowTrap is a high-interaction SSH Honeypot built for forensic intelligence. It captures real-time data from attackers, including passwords, IP addresses, and executed commands.

The backend uses Python and Paramiko to handle SSH traffic, while the frontend visualizes the data using Flask and Chart.js.
