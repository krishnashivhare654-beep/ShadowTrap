📖 ABOUT PROJECT

ShadowTrap is a high-interaction SSH Honeypot developed to monitor and analyze unauthorized access attempts. Unlike simple firewalls, ShadowTrap lures attackers into a controlled environment where their every move is logged for forensic study.

Built with Python, Flask, and Paramiko, it provides real-time threat intelligence and visualizes attack patterns to improve defensive strategies.

❓ HOW TO USE

To test the forensic engine, initiate an attack from a remote terminal (like Termux):

ssh root@10.16.71.2 -p 2222

Use any password to enter. Once inside, try commands like 'ls', 'whoami', or 'rm'. Your session will be recorded and displayed in the monitor feed above.
