📖 PROJECT OVERVIEW

ShadowTrap is a high-interaction SSH Honeypot. It creates a deceptive environment to lure attackers, logging every command and credential used for forensic study. Built with Python and Flask, it provides real-time threat intelligence through this automated dashboard.

❓ HOW TO TEST

Run this command from any terminal to trigger a log:

ssh root@10.16.71.2 -p 2222

Once connected, try running commands like 'whoami' or 'ls'.
