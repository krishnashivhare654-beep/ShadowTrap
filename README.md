## 🎯 What is ShadowTrap?

ShadowTrap is a **fake SSH server** that mimics a real Linux machine to deceive attackers. Every credential, command, and IP address is silently logged for security analysis.

## ⚡ Features

- 🎭 Realistic SSH Banner (OpenSSH 7.4)
- 🔐 Credential Capture
- 💻 Command Logging
- 📊 Live Dashboard
- 🗄️ SQLite Database
- 🌍 IP Tracking
